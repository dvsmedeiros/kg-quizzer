pergunta;resposta
O que é o sistema imunológico?
Uma rede de processos biológicos que protege contra doenças.
Quais são os dois principais subsistemas do sistema imunológico?
Sistema imunológico inato e adaptativo.
O que é o sistema imunológico inato?
Resposta preconfigurada a grupos amplos de situações.
Como o sistema imunológico adaptativo funciona?
Aprende a reconhecer moléculas previamente encontradas.
O que são defensinas?
Peptídeos antimicrobianos.
O que é imunidade adquirida e como se relaciona com a vacinação?
Memória imunológica que melhora a resposta a patógenos.
O que pode causar disfunção do sistema imunológico?
Doenças autoimunes, inflamatórias e câncer.
O que é imunodeficiência?
Sistema imunológico menos ativo que o normal.
